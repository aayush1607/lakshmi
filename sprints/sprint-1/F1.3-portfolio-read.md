# F1.3 — Portfolio Read-Only View

## One-liner
`/portfolio` shows the user's current Zerodha holdings as a clean table.

## User story
> As a logged-in user, I want to see all my holdings at a glance inside Lakshmi, so I trust the tool has the right view of my money before I ask it anything.

## Expected behaviour (PM view)
- User types `/portfolio` (aliases: `/p`, `/holdings`).
- Lakshmi pulls the latest holdings from Zerodha and shows:
  ```
  ━━━ HOLDINGS ━━━  (as of 17 Apr 2026, 10:42 IST)

   SYMBOL      QTY     AVG ₹      LTP ₹     P&L ₹      P&L %    WEIGHT
   TCS          15    3,420      3,980   +8,400      +16.4%     14.2%
   HDFCBANK     42    1,510      1,642   +5,544       +8.7%     12.1%
   INFY         30    1,420      1,380     -600       -1.4%      7.3%
   …

   ━━━ TOTALS ━━━
   Invested     ₹ 9,85,000
   Current      ₹ 11,42,300
   Day P&L      ₹    +4,210 (+0.37%)
   Overall P&L  ₹ +1,57,300 (+15.97%)

   Source: Zerodha Kite (live) · refreshed 3s ago
  ```
- Columns sort by weight descending by default.
- Flags: `/portfolio --by pnl`, `/portfolio --by symbol`.
- If the session is expired, Lakshmi says so and points to `/login`.
- If markets are closed, the note reads *"post-close quote"* rather than "live".

## Acceptance criteria
1. First-time call completes in under 3 seconds on a normal connection.
2. Numbers match what Kite web shows at the same moment (spot-check on 5 symbols).
3. No write operations are ever sent to Zerodha — read-only guaranteed.
4. Empty portfolio renders a friendly "No holdings yet" message (no crash).
5. Works offline from cache (F1.4) with a clear "cached data, N minutes old" banner.

## Out of scope
- Positions (intraday), orders, funds, mutual funds — holdings only.
- Any analysis/verdict (that comes in Sprint 2).
- Editing or placing trades.

## Dependencies
- F1.1 (REPL)
- F1.2 (Login)
- F1.4 (Local store — for caching)

## Demoable in
Login, run `/portfolio`, show the table, toggle sort flag, disconnect wifi, run `/portfolio` again, see cached view with staleness banner.
