# F2.2 — `/ta <ticker>` Technical Snapshot

## One-liner
One command gives a complete, grounded technical verdict on any NSE stock.

## User story
> As a user sizing up a trade, I want to type `/ta TCS` and immediately see whether the chart is bullish, bearish, or mixed — with the exact signals that drove the call.

## Expected behaviour (PM view)
- User runs `/ta HDFCBANK`.
- Output follows the universal verdict format and contains:
  - **Verdict**: 🟢 Bullish / 🟡 Mixed / 🔴 Bearish — one line.
  - **Signals block** with four rows, each with an arrow + short label:
    - Trend (DMA alignment 20/50/200)
    - Momentum (RSI 14)
    - Volume (relative to 20-day average)
    - Pattern (best-detected pattern + confidence, if any)
  - **Key levels**: nearest resistance, nearest support, suggested stop-loss with explanation.
  - **Sources**: NSE bhavcopy (Tier 1) + "computed locally".
- Works fully offline once F2.1 has run.
- Runs in under 1 second for any cached ticker.
- Unknown tickers get a polite "symbol not found in NSE universe" message with suggested close matches.

## Acceptance criteria
1. The four signal rows always show a direction (▲ / ▼ / ◆) — never blank.
2. Numbers (RSI, volume ratio) match a manual calculation on the same OHLCV within 1%.
3. Support / resistance values fall within the 6-month price range of the stock.
4. A freshly listed stock with < 200 trading days is flagged "insufficient history" instead of failing.
5. No external API call is made if data is cached.

## 🏰 Moat contribution
Indicators computed in pure Go, in-tree, with unit tests against known reference values. This matters because Claude / ChatGPT cannot run a reproducible, auditable RSI or DMA on demand — they'll hallucinate a number. Deterministic, locally-computed TA is a small but real defensibility layer; it's also what makes alerts (F3.4) and backtests (F3.2) trustworthy.

## Out of scope
- Chart rendering (that's F2.3).
- Intraday / minute-level TA.
- Options-based signals.
- Custom indicator configs (RSI period, etc. — fixed defaults in v1).

## Dependencies
- F1.6 (Response shaper)
- F2.1 (NSE data)

## Demoable in
Run `/ta` on 3 tickers with clearly different regimes — a bull trend, a downtrend, and a consolidation — and show that the verdicts and signals make sense.
