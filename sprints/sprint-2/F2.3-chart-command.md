# F2.3 — `/chart <ticker>` In-Terminal Chart

## One-liner
A readable price chart rendered directly in the terminal — no browser, no external tool.

## User story
> As a terminal-native user, I want to eyeball price action without leaving my shell, so I can stay in flow.

## Expected behaviour (PM view)
- `/chart TCS` renders a daily candlestick (or line-fallback) chart of the last 3 months.
- Timeframe flags: `--1m`, `--3m`, `--6m`, `--1y`, `--5y` (default: 3m).
- Optional overlays:
  - `--dma 20,50,200` adds moving-average lines.
  - `--levels` overlays support / resistance (from F2.2).
  - `--volume` adds a volume pane at the bottom.
- Axis labels show dates at the bottom and price ticks on the right.
- Current price, day change, and volume vs average appear as a one-line header.
- If the terminal supports Kitty graphics protocol, a higher-fidelity image is drawn; otherwise Unicode blocks are used.
- Works offline from cached data.

## Acceptance criteria
1. Chart fits in 80 columns × 24 rows minimum without breaking.
2. Candles correctly show green when close > open, red when close < open.
3. Switching timeframes re-renders in < 300 ms.
4. Overlays can be combined (`--dma 50,200 --volume`) without layout breaking.
5. Running `/chart` on a non-existent ticker shows the same "not found" message as `/ta`.

## Out of scope
- Intraday charts.
- Drawing tools, annotations, saving images.
- Comparison charts (that's the `/compare` command — Sprint 2 stretch or later).
- Interactive cursor/crosshair.

## Dependencies
- F2.1 (NSE data)

## Demoable in
Show `/chart TCS`, `/chart TCS --1y --dma 50,200 --volume`, and `/chart HDFCBANK --levels` side-by-side.
